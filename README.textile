h1. categorize

Text categorization library.
https://rubygems.org/gems/categorize

h2. Notes

Rate limiting is enforced as per the Blekko API usage guidelines.

h2. Installation

* install the gem

@gem install categorize@

h2. Example

@>> Categorize.make_model('nanotubes', documents)@
@=> [ 'nanotechnology' => [0, 3, 5], ... ]@
